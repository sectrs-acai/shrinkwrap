########
Commands
########

For documentation on the commands that shrinkwrap supports, add ``--help`` to
the command line.

For top-level help:

.. code-block:: shell

  shrinkwrap --help

For help on a specific command:

.. code-block:: shell

  shrinkwrap inspect --help
  shrinkwrap build --help
  shrinkwrap clean --help
  shrinkwrap run --help
  shrinkwrap process --help
